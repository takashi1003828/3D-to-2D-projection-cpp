# 3D to 2D Projection (C++ / SDL2)

3D空間の座標を2Dスクリーンに変換して描画するプログラムを、ゲームエンジンを使用せずにC++とSDL2のみでゼロから実装しました。

## 概要
* **開発言語:** C++
* **ライブラリ:** SDL2
* **主な機能:** * 3D座標から2Dスクリーンへの透視投影（Perspective Projection）
  * キーボード入力によるリアルタイムなX軸・Y軸の回転処理
  * ダブルバッファリングを用いたちらつきのない描画

## アピールポイント（技術的な工夫）
1. **数学的ロジックの自作**
   外部の3Dライブラリに依存せず、頂点座標の回転行列（三角関数）や、Z除算による遠近法の計算ロジックを自ら実装しています。
2. **堅牢性の確保**
   投影計算時のゼロ除算（Z=0）を防ぐためのフェイルセーフを実装し、クラッシュを防ぐ設計にしています。
3. **マジックナンバーの排除と保守性**
   画面サイズや視野角（FOV）、描画サイズなどをすべて `constexpr` で定数化し、後から調整がしやすい保守性の高いコードを意識しました。

## 今後の課題・展望
現在は画家のアルゴリズムやZバッファを実装していないため、ポリゴンの描画順序に課題があります。今後は深度テストの概念を導入し、より正確な3Dレンダリングへ拡張していく予定です。
